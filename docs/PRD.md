# PRD — OTT 큐레이션 앱

> 최종 수정: 2026-02-28
> 상태: 초안 (v0.1)

---

## 1. 제품 개요

### 비전
한국 OTT 구독자가 "오늘 뭐 볼까?"를 고민하는 시간을 없애준다.
5개 플랫폼(넷플릭스·티빙·쿠팡플레이·웨이브·왓챠)의 콘텐츠를 한 곳에서 보고, Claude AI에게 자연어로 "비오는 날 혼자 볼 영화"라고 말하면 딱 맞는 추천을 받는다.

### 핵심 문제
1. **플랫폼 분산**: 어떤 OTT에 뭐가 있는지 모름 → 매번 앱을 돌아다님
2. **추천의 맥락 부재**: 기존 알고리즘은 "비슷한 콘텐츠"만 추천 → 지금 내 기분/상황과 무관
3. **AI 콘텐츠 전용 공간 없음**: AI 생성 숏폼이 일반 SNS에 섞여 품질 판단 어려움

### 타겟 사용자
- 1차: 한국 OTT 2개 이상 구독 중인 20~40대
- 2차: AI 생성 콘텐츠에 관심 있는 크리에이터

---

## 2. 핵심 기능 (Epics)

### Epic 1. OTT 통합 브라우저
5개 OTT의 콘텐츠 가용성을 실시간 동기화해서 "내가 구독한 플랫폼에서 볼 수 있는 것만" 필터링.

**핵심 사용자 스토리**
- 사용자는 구독 중인 OTT를 선택하면, 그 플랫폼에서만 볼 수 있는 콘텐츠 목록을 본다.
- 사용자는 콘텐츠 상세 화면에서 어떤 OTT에서 볼 수 있는지 한눈에 확인하고 바로 이동한다.
- 사용자는 제목·출연진으로 한국어 퍼지 검색을 한다.

### Epic 2. AI 자연어 큐레이션 챗
Claude AI에게 자연어로 상황을 설명하면 pgvector 의미 검색 + Claude 분석을 결합해 추천.

**핵심 사용자 스토리**
- 사용자는 "주말에 가족이랑 볼 만한 것"처럼 자유롭게 입력하면 맥락에 맞는 추천 목록을 받는다.
- 사용자는 AI가 이 추천을 왜 했는지 간단한 이유 문장을 함께 본다.
- 사용자는 추천 결과에서 바로 평점을 남길 수 있다.

### Epic 3. 개인화 홈 큐레이션
사용자의 평점 데이터를 pgvector 임베딩으로 변환해 취향 벡터를 구성, 매일 새벽 사전 계산된 개인화 섹션을 홈에 표시.

**핵심 사용자 스토리**
- 사용자는 홈에서 "당신을 위한 오늘의 선택" 같은 개인화 섹션을 본다.
- 평점이 쌓일수록 추천이 정교해진다.

### Epic 4. AI 콘텐츠 관 (숏폼 피드)
AI 생성 숏폼 전용 공간. 업로드 시 Claude Vision이 자동 모더레이션하고, 피드 스코어 알고리즘으로 노출 순서를 결정.

**핵심 사용자 스토리**
- 크리에이터는 AI로 만든 영상(최대 60초)을 업로드한다.
- 시청자는 TikTok 방식의 세로 피드에서 AI 숏폼을 소비한다.
- 업로드 후 수 분 내에 자동 모더레이션 결과를 받는다.

### Epic 5. 커뮤니티 (평점·리뷰·컬렉션)
- 0.5점 단위 평점 (조화평균 집계)
- 스포일러 토글 리뷰
- 공유 가능한 컬렉션

## 3. 비기능 요구사항

| 항목 | 목표 |
|------|------|
| API 응답시간 | p95 < 300ms (일반 목록), AI 큐레이션 챗 < 5s |
| 가용성 | 99.5% (Cloud Run 자동 스케일) |
| 콘텐츠 동기화 주기 | 매일 01:00 KST |
| 임베딩 갱신 | 매일 05:00 KST |
| 개인화 사전 계산 | 매일 03:00 KST |
| 피드 스코어 갱신 | 15분마다 |
| 파일 업로드 제한 | 최대 60초 / 500MB |
| 모더레이션 SLA | 업로드 후 5분 내 자동 처리 |

## 4. 아키텍처 결정

| 영역 | 선택 | 이유 |
|------|------|------|
| 앱 | Flutter | iOS/Android 단일 코드베이스 |
| 인증 | Firebase Auth | 카카오·Apple·Google 소셜 로그인 통합 |
| API | Cloud Run (Node.js/TS) | 서버리스 자동 스케일 |
| DB | Cloud SQL PostgreSQL 15 | pgvector + pg_trgm 지원 |
| AI 추천 | Claude API + Voyage AI | 한국어 임베딩 최적 모델 |
| 스토리지 | Cloud Storage | 대용량 영상 직접 업로드 (Signed URL) |
| 콘텐츠 데이터 | JustWatch GraphQL + TMDb | 한국 OTT 가용성 실시간 |
